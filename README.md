# Line-Following-Robot

Self-navigating line-following robot using sensors and a microcontroller

---

## Objectives
- Build an autonomous robot
- Implement line-following logic
- Improve navigation accuracy
- Apply control system concepts

---

## How It Works
The robot uses line sensors to detect the path.
Based on sensor input, the microcontroller adjusts motor movement to keep the robot on track.

- If the robot deviates → it corrects direction
- If centered → it moves forward

---

## Components Used
- Microcontroller (Arduino Uno)
- Line Sensors (IR sensors)
- Motor Driver (L298N)
- DC Motors
- Wheels & Chassis
- Power Supply 5V

---

## Features
- Autonomous navigation
- Real-time path correction
- Simple control algorithm
- Stable movement
